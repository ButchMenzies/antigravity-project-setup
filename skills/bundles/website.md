# Website Bundle

Skills for static or content sites (marketing, portfolio, blog, landing pages).
Typically: Astro + Tailwind or Next.js static export.

## Skills

- build-page
- build-component
- seo-audit
- deploy-vercel

## Recommended MCPs

| MCP | Purpose | Required |
|-----|---------|----------|
| supabase | Dynamic content, contact forms, CMS data | No — optional for static sites |
| vercel | Deployment, preview environments | No — manual deploy as fallback |
