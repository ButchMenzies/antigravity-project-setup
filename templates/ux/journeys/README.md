# Journey Maps

This folder contains per-feature emotional journey maps.

Create a new file for each major feature: `[feature-name].md`

## Journey Map Template

```markdown
# Journey: [Feature Name]

## Arrival
What emotion does the user bring to this feature?

## First Value
How quickly do they feel "this is for me"? (Target: < 30 seconds)

## Investment
What makes them put something of themselves into it?

## Reward
What's the "I did it" moment?

## Return
Why do they come back to this feature?
```
